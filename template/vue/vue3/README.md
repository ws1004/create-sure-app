# Vue 3 + TypeScript + Vite

- 🎻**Vue3**,组合式 API,更加优雅的开发应用程序.
- 🥊**TypeScript**,增加类型检查系统,增强应用程序健硕性.
- ⚡️**Vite**,下一代前端构建工具,闪电般的速度.
- 🏄**一键配置**导入依赖,无需 import.
- 🪂**component**目录下组件无需引入直接使用
- 🧩**UnoCss**原子化 CSS,极大增加编写 CSS 的体验.

## 使用

### 1.安装依赖

```sh
pnpm install
```

### 2.启动

```sh
pnpm run dev
```
